# Abeml™ Licensing FAQ

> This Markdown is provided for readability. The **authoritative** version is
> [`Abeml-Licensing-FAQ.pdf`](Abeml-Licensing-FAQ.pdf). See the
> [EULA](EULA.md) and [Abe™ Runtime Redistribution License](../runtime/ABE-RUNTIME-REDISTRIBUTION-LICENSE.md)
> for the controlling terms.

### Do I retain ownership of my code?
Yes. You retain ownership of all software and other works you create using Abeml™.

### Can I sell software built with Abeml™?
Yes. Commercial sale, licensing, and distribution of software built with Abeml™ are
permitted.

### Can I deploy Abe™ applications without runtime royalties?
Yes, for the **Free Runtime** (the Level-1 runtime that ships with the free abepro
edition). E-Tools does not charge runtime royalties, deployment fees, per-user fees, or
per-installation fees for it. The **Full Runtime** (Level 1 + 2 / ML) is different: it is
commercial, ships only with the abeml edition, and is governed by the EULA — see below.

### Can I redistribute the Abe™ Runtime royalty-free with my application?
The **Free Runtime**, yes: applications may be distributed together with it in accordance
with the Abe™ Runtime Redistribution License (static or dynamic linking). The **Full
Runtime** (the ML runtime) is not covered by the ARRL — it is not free, is not available
outside abeml, and may only travel embedded in applications you build with abeml, under
the EULA.

### Do I need a paid license to use Abeml™ commercially?
Yes. Commercial use of the Abeml™ compiler requires a valid commercial license unless
otherwise authorized by E-Tools.
